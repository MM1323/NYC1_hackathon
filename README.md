# Node website

## How to get started:
* `cd NYC1_hackathon`
* `npm install`
* `npm run dev` to start the development site
* `npm run test` to run your tests
* `npm start` to build a production `dist` folder transpiled to `ES5` with [Babel](https://babeljs.io/) and start the production server
